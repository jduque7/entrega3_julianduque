# Entrega 3 - Joyería JDuque

## Descripción
joyería desarrollado en Django. Incluye gestión de modelos, formularios y búsqueda.

## Funcionalidades
* **Modelos:** Categorías, Materiales y Joyas.
* **Formularios:** Carga de datos para las 3 clases.
* **Búsqueda:** Buscador de joyas por nombre.
